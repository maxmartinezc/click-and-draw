# Click And Draw
The "click and draw" project, get a term and draw it!

## M.E.A.N App
MongoDB, Express, Angular 2 & NodeJS

### Installation

#### Install bower components
- Bootstrap
bower install bootstrap#v4.0.0-alpha.5

- Font awesome
bower install components-font-awesome --save

#### RUN Steps
##### Server
run nodemon app.js

##### Client
cd client
run npm start

#### Browser App
http://localhost:3000
